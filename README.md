# Lab_6

Задача 1. Загрузите данные из xml-файлов и рассчитайте общую статистику по всем тренировкам.
Для этого требуется:
• реализовать функцию load_users_data() для загрузки данных о пользователях;
• реализовать функцию load_workouts_data() для загрузки данных о тренировках;
• реализовать функцию get_stats(users, workouts) для расчета общей статистики

Задача 2. Найдите ТОП-3 самых активных пользователей, выведите распределение данных по
типам тренировок, а также выполните детальный анализ тренировок по имени пользователя. Для
этого требуется:
• реализовать функцию analyze_user_activity(users), которая для каждого пользователя
посчитает общее количество тренировок, общее количество калорий, суммарное время
тренировок (в часах) и выведет 3-х самых активных пользователей
• реализовать функцию analyze_workout_types(workouts), которая выведет распределение
по типам тренировок
• реализовать функцию find_user_workouts(users, user_name), которая находит и возвращает
все тренировки пользователя по имени
• реализовать функцию analyze_user(user, workouts), которая выводит детальный анализ
тренировок пользователя по имени

Задача 3. Выполните визуализацию данных, построив следующие диаграммы:
• круговая диаграмма типов тренировок
• столбчатая диаграмма активности пользователей
• столбчатая диаграмма эффективности тренировок
• сравнительная диаграмма пользователей
